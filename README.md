# MomoPlayer
a simple RTSP player.

windows only, video stream only, audio is not supported yet.

LAN latency is between 200 and 300 milliseconds.

### development environment：
* WIN10
* VS2017
* Qt5.12.9
* FFmpeg4.3.1 64bit


# 梦梦播放器
简单的RTSP播放器

windows平台，纯视频流播放，暂不支持音频。

局域网延迟在200到300毫秒。

### 开发环境：
* WIN10
* VS2017
* Qt5.12.9
* FFmpeg4.3.1 64位
